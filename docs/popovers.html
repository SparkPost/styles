Popovers

Popover sizes

Define popover sizes with .popover--s, .popover--m, or .popover--l. These set the width of the popover. You can also specify width manually.

I am a very very very very very very very very very large popover 
            
                <button class="button popover__group">

                  ...

                  <span class="popover popover--s"> ... </span>

                </button>



                <button class="button popover__group">

                   ... 

                  <span class="popover"> ... </span> // or popover--m

                </button>



                // Another way to structure these

                <div class="popover__group">

                    <button class="button"> ... </button>

                    <span class="popover popover--l"> ... </span>

                </div>
Popover alignment

Use the modifiers .popover--leftAligned and .popover--rightAligned to align popovers to the left or right. These only work with top or bottom positioning.

            

              <button class="button button--muted button--full popover__group">

                  ...

                  <span class="popover popover--leftAligned"> ... </span>

              </button>



              <button class="button button--muted button--full popover__group">

                  ...

                  <span class="popover popover--bottom popover--rightAligned"> ... </span>

              </button>
Popover placement

Use the modifiers .popover--left, .popover--right, and .popover--bottom to change positioning around the trigger. Top is used by default.

Click Me I am a left positioned popover.
Click Me I am right positioned popover.
Click Me I am bottom positioned popover. 
            

                <a class="popover__group" ... >

                    ...

                    <span class="popover popover--left"> ... </span>

                </a>



                <a class="popover__group" ... >

                    ...

                    <span class="popover popover--right"> ... </span>

                </a>



                <a class="popover__group" ... >

                    ...

                    <span class="popover popover--bottom"> ... </span>

                </a>
Open state

If you need to activate the popover through hover, add the .popover__hoverTrigger class to the wrapper group. Toggle visibility of the popover with the .is-open class on the popover itself.

  • Must be at least 8 characters.
  • Must be at least one number and one letter.
 
            

              <div class="input__group popover__group popover__hoverTrigger">

                  <input class="input__text" ... />

                  <span class="popover popover--l popover--leftAligned popover--bottom is-active">

                      ...

                  </span>

              </div>
