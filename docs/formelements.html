Form elements

Fieldset Components

There are 3 main components: .fieldset, .label, and .input_group. Fieldsets by default fill in the width of its parent, use the grid to size and place inputs where you want them.

Input groups have 3 possible children for text and textarea inputs: the input, a font awesome icon, and a span for errors.

Example of all the components put together:

This is a hidden error
This is a visible error 
            
              <div class="flex">

                  <div class="col-xs-12 col-md-6">

                      <fieldset class="fieldset">


                          <label class="label" for="foo">...</label>


                          <div class="input__group">

                              <input class="input__text" id="foo" type="text" ... />

                              <span class="input__error">This is a hidden error</span>

                          </div>


                      </fieldset>

                  </div>

                  <div class="col-xs-12 col-md-6">

                      <fieldset class="fieldset">


                          <label class="label" for="bar">...</label>


                          <div class="input__group has-error">

                              <input class="input__text" id="bar" type="text" ... />

                              <span class="input__error">This is a visible error</span>

                          </div>


                      </fieldset>

                  </div>

              </div>
Input sizes

Use the modifiers input__text--s, input__text--m and input__text--l to adjust font-size and padding within text inputs.

            
              <input class="input__text input__text--s" type="text" ... />

              <input class="input__text input__text--m" type="text" ... />

              <input class="input__text input__text--l" type="text" ... />
Disabled

Use the class .is-disabled and the disabled attribute to disable an input

            
              <input class="input__text is-disabled" ... disabled/>
Input with error

Use the class .has-error on the input group to display its error state.

This is an error message 
            
              <fieldset class="input__fieldset has-error">

                <input class="input__text" ... />

                <span class="input__error">...</span>

              </fieldset>
Input with icon

Append your desired font awesome <i> to the input group, and add the .input__group--icon modifier to the input group.

This is an error message 
            
              <div class="input__group input__group--icon">

                  <input class="input__text" type="text" ... />

                  <i class="fa fa-envelope"></i>

              </div>
Textarea

Use the modifiers input__textarea--s, input__textarea--m and input__textarea--l to adjust font-size and padding within the textarea.

Icons work with these too.

This is an error message 
            
              <div class="input__group input__group--icon">

                  <textarea class="input__textarea input__textarea--l" ...></textarea>

                  <i class="fa fa-commenting"></i>

              </div>
Select

Selects require an additional span tag for the dropdown arrow. <span class="input__selectArrow"></span>

Like the other inputs, use the modifiers input__select--s, input__select--m and input__select--l to adjust font-size and padding.

And icons work with these too.

            
              <div class="input__group input__group--icon">

                  <select class="input__select">

                      ...

                  </select>

                  <span class="input__selectArrow"></span>

                  <i class="fa fa-plane"></i>

              </div>
