TODO: finish

The grid uses a responsive 12 column system built on flexbox. Wrap columns with the grid namespace <div class="flex">. Use them like you would use a row, but because this is flexbox, they aren't necessarily rows and could be columns, ya know? The grid takes full the width of its parent, you can place these anywhere, ie. within a container, modal, or within another grid column.

A column without a declared width will figure it out by itself.
