Box model helpers

Spacing can be applied using the margin and padding helpers. Both margin and padding share the same scale. There are 8 spacing sizes available through these suffixes; --xxs, --xs, --sm, --md, --lg, --xl, --xxl, and --none.

--md outputs 1 rem, and is used as the baseline for vertical rhythm.

These aren't responsive.. yet!

 
 
 
 
 
 
            
              // padding

              <div class="padding--none"></div>

              <div class="padding--xxs"></div>

              <div class="padding--xs"></div>

              <div class="padding--sm"></div>

              <div class="padding--md"></div>

              <div class="padding--lg"></div>

              <div class="padding--xl"></div>

              <div class="padding--xxl"></div>



              // margin

              <div class="margin--none"></div>

              <div class="margin--xxs"></div>

              <div class="margin--xs"></div>

              <div class="margin--sm"></div>

              <div class="margin--md"></div>

              <div class="margin--lg"></div>

              <div class="margin--xl"></div>

              <div class="margin--xxl"></div>
 
 
 
 
            
              // padding

              <div class="paddingTop--md"></div>

              <div class="paddingRight--md"></div>

              <div class="paddingBottom--md"></div>

              <div class="paddingLeft--md"></div>



              // margin

              <div class="marginTop--md"></div>

              <div class="marginRight--md"></div>
