Panels

This container is a panel!

.panel__heading holds the panel title, and .panel__body holds the content. You can omit the heading if needed.

The modifier .panel--accent adds an orange accent bar to the top of the panel.

Hello World
Foo
    
      <div class="panel">

          <div class="panel__heading"><h5>...</h5></div>

          <div class="panel__body">...</div>

          <div class="panel__body">...</div>

      </div>
Panel widths

Panels fill in the width of their parent container. Here's an example of how to use panels with our grid system.

Hello
World
    
      <div class="flex">

            <div class="col-xs-8">

                  <div class="panel panel--accent">

                        <div class="panel__body">...</div>

                  </div>

            </div>

            <div class="col-xs-4">

                  <div class="panel">

                        <div class="panel__body">...</div>

                  </div>

            </div>

      </div>
