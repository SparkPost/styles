Tooltips

Tooltip sizes

Define tooltip sizes with .tooltip--s, .tooltip--m, or .tooltip--l. These set the width of the tooltip.

            
                <button class="button tooltip__group tooltip__hoverTrigger">

                  ...

                  <span class="tooltip tooltip--s"> ... </span>

                </button>



                <button class="button tooltip__group tooltip__hoverTrigger">

                   ... 

                  <span class="tooltip"> ... </span> // or tooltip--m

                </button>



                <button class="button tooltip__group tooltip__hoverTrigger">

                   ... 

                  <span class="tooltip tooltip--l"> ... </span>

                </button>
Tooltip alignment

Use the modifiers .tooltip--leftAligned and .tooltip--rightAligned to align tooltips to the left or right. These only work with top or bottom positioning.

            

              <button class="button button--muted button--full tooltip__group tooltip__hoverTrigger">

                  ...

                  <span class="tooltip tooltip--leftAligned"> ... </span>

              </button>



              <button class="button button--muted button--full tooltip__group tooltip__hoverTrigger">

                  ...

                  <span class="tooltip tooltip--bottom tooltip--rightAligned"> ... </span>

              </button>
Tooltip colors

Use the modifiers .tooltip--orange and .tooltip--dark to change background color. White is used by default.

            

              <label class="label">

                  Label 

                  <span class="tooltip__group tooltip__hoverTrigger">

                      <i class="fa fa-question-circle"></i>

                      <span class="tooltip tooltip--dark">I am a dark tooltip.</span>

                  </span>

              </label>



              <label class="label">

                  Label 

                  <span class="tooltip__group tooltip__hoverTrigger">

                      <i class="fa fa-question-circle"></i>

                      <span class="tooltip tooltip--orange">I am an orange tooltip.</span>

                  </span>

              </label>
Tooltip placement

Use the modifiers .tooltip--left, .tooltip--right, and .tooltip--bottom to change positioning around the trigger. Top is used by default.

Hover me I am a left positioned tooltip.
Hover me I am right positioned tooltip.
Hover me I am bottom positioned tooltip. 
            

                <a class="tooltip__group tooltip__hoverTrigger" ... >

                    ...

                    <span class="tooltip tooltip--left"> ... </span>

                </a>



                <a class="tooltip__group tooltip__hoverTrigger" ... >

                    ...

                    <span class="tooltip tooltip--right"> ... </span>

                </a>



                <a class="tooltip__group tooltip__hoverTrigger" ... >

                    ...

                    <span class="tooltip tooltip--bottom"> ... </span>

                </a>
Active state

If you need to activate the tooltip through means other than hover (eg, through javascript), omit the .tooltip__hoverTrigger class. Toggle visibility of the tooltip with the .is-active class on the tooltip itself. You will still need the group class to position the tooltip.

  • Must be at least 8 characters.
  • Must be at least one number and one letter.
 
            

              <div class="input__group tooltip__group">

                  <input class="input__text" ... />

                  <span class="tooltip tooltip--l tooltip--dark tooltip--leftAligned tooltip--bottom is-active">

                      ...

                  </span>

              </div>
