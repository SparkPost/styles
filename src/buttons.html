Buttons

Button colors

Define button colors with .button--blue or .button--muted. Orange is used with no modifier class.

            
              <button class="button">...</button>

              <button class="button button--blue">...</button>

              <button class="button button--muted">...</button>

              // or

              <a class="button">...</a>
Button sizes

Define button font and padding sizing with .button--l, .button--m, or .button--s. Medium is used with no modifier class.

            
              <button class="button button--l">...</button>

              <button class="button button--m">...</button>

              <button class="button button--s">...</button>
Full width

.button--full expands the button to 100% of its parent's width.

            
              <button class="button button--blue button--full">...</button>
Button states

Use the .is-pressed class to show an active state. Visually the same as &:active styles.

.is-disabled and the disabled attribute disables pointer events.

            
              <button class="button is-pressed">...</button>

              <button class="button button--blue is-disabled" disabled >...</button>
Button with icon
            
              <button class="button button--muted"><i class="fa fa-trash"></i> Button</button>

              <button class="button button--icon"><i class="fa fa-plus"></i></button>
Button group

Wrap two or more buttons within .button-group.

            
              <div class="button-group">

                  <button class="button button--muted">...</button>

                  <button class="button button--muted">...</button>

                  <button class="button button--muted" disabled>...</button>

              </div>
Button accents
            
              <button class="button button--muted button--blueAccent">Button<i class="fa fa-close paddingLeft--xs"></button>

              <button class="button button--muted button--tealAccent">...</button>

              <button class="button button--muted button--greenAccent">...</button>

              <button class="button button--muted button--magentaAccent">...</button>
