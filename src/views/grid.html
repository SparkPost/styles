Grid

The grid uses a responsive 12 column system built on flexbox. Wrap columns with your grid namespace <div class="row">. Rows are not needed, but if you wish to force the next column down, simply close the grid and reopen. The grid takes full the width of its parent, you can place these anywhere, ie. within a container, modal, or within another grid column.
