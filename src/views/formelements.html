Input sizes

Use the modifiers --s, --m and --l to adjust font-size and padding within text inputs. Use --full to expand its width to the width of its parent.

            
              <input class="input--text input--s" type="text" ... />

              <input class="input--text input--m" type="text" ... />

              <input class="input--text input--l" type="text" ... />

              <input class="input--text input--l input--full" type="text" ... />
Disabled

Use the class .is-disabled and the disabled attribute to disable an input

            
              <input class="input--text is-disabled" ... disabled/>
