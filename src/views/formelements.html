Form elements

Fieldset Components

Fieldsets have 3 children: an input, a label, and a span for errors. By default, Fieldsets are inline blocks and input width is set by the browser, use .input__fieldset--full to expand the fieldset and its input to the width of its parent.

Example of all the components put together:

This is a hidden error
This is a visible error 
            
              <div class="row">

                  <div class="col-xs-12 col-md-6">

                      <fieldset class="input__fieldset input__fieldset--full">

                          <label class="input__label" for="foo">Label</label>

                          <input class="input__text" id="foo" type="text" ... />

                          <span class="input__error">This is a hidden error</span>

                      </fieldset>

                  </div>

                  <div class="col-xs-12 col-md-6">

                      <fieldset class="input__fieldset input__fieldset--full has-error">

                          <label class="input__label" for="bar">Label</label>

                          <input class="input__text" id="bar" type="text" ... />

                          <span class="input__error">This is a visible error</span>

                      </fieldset>

                  </div>

              </div>
Input sizes

Use the modifiers input__text--s, input__text--m and input__text--l to adjust font-size and padding within text inputs.

            
              <input class="input__text input--s" type="text" ... />

              <input class="input__text input--m" type="text" ... />

              <input class="input__text input--l" type="text" ... />
Disabled

Use the class .is-disabled and the disabled attribute to disable an input

            
              <input class="input__text is-disabled" ... disabled/>
Input with error

Use the class .has-error on the fieldset to display its error state. Note: a label is required to position the error message correctly. (You should always be using labels anyway)

This is an error message 
            
              <fieldset class="input__fieldset has-error">

                <input class="input__text" ... />

              </fieldset>

Textarea

Use the modifiers input__textarea--s, input__textarea--m and input__textarea--l to adjust font-size and padding within the textarea.

This is an error message 
            
              <textarea class="input__textarea" ... ></textarea>
Select

Selects require an additional span tag for the dropdown arrow. <span class="input__selectArrow"></span>

Like the other inputs, use the modifiers input__select--s, input__select--m and input__select--l to adjust font-size and padding.

This is an error message 
            
              <fieldset class="input__fieldset input__fieldset--full">

                  <label class="input__label">Label</label>

                  <select class="input__select">

                      ...

                  </select>

                  <span class="input__selectArrow"></span>

                  <span class="input__error">This is an error message</span>

              </fieldset>
