Buttons

Button colors

Define button colors with .button--blue or .button--muted. Orange is used with no modifier class.

            
              <button class="button">...</button>

              <button class="button button--blue">...</button>

              <button class="button button--muted">...</button>
Button sizes

Define button font and padding sizing with .button--l, .button--m, or .button--s. Medium is used with no modifier class.

            
              <button class="button button--full">...</button>

              <button class="button button--l">...</button>

              <button class="button button--m">...</button>

              <button class="button button--s">...</button>
Other options

.button--full expands the button to 100% of its parent's width. .button--disabled or the disabled attribute disables pointer events.

            
              <button class="button button--blue button--full">...</button>


              <button class="button button--blue button--disabled">...</button>

              // or

              <button class="button button--blue" disabled>...</button>
