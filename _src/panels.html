Panels

This container is a panel!

.panel__heading holds the panel title, .panel__body holds the content, and .panel__footer creates a slightly darker section with an inset box shadow

The modifier .panel--accent adds an orange accent bar to the top of the panel. .panel__body--darkGray and .panel__body--lightGray are also available to change the background color of a section.

Tabs

Append a ul.panel__tabList element to create a tab group. Use li.panel__tabItem > a.panel__tabLink to define the individual tabs.

Add .is-active to .tabItem to use its active state.
